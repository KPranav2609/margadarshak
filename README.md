# 🚀 MargaDarshak

AI-powered placement preparation platform that helps students track progress, improve weak areas, and receive personalized mentorship.

---

## 🔥 Features

* ✅ Task Management System (Coding, Aptitude, Interview)
* 📊 Dashboard Analytics (progress tracking)
* 🤖 AI Mentor (personalized guidance)
* 🔐 Secure Authentication (JWT-based)
* 📈 Category-wise performance tracking

---

## 🧠 Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

### Frontend

* React (in progress)

### AI Integration

* OpenRouter API (LLM-based mentor system)

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
OPENAI_API_KEY=your_api_key
```
